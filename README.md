# Tailwind 기반 Vue 디자인 시스템

## 🎯 개요
- 일관된 UI/UX를 위한 디자인 시스템
- Tailwind CSS 커스터마이징 + Vue 컴포넌트 기반

## 📁 구성
- `BaseButton`, `BaseInput`, `BaseCard` 등의 재사용 컴포넌트
- `tailwind.config.js`에 정의된 디자인 토큰(color, spacing 등)

## 🧩 주요 컴포넌트
- 버튼: primary / secondary / danger
- 입력창: 기본 입력창, 포커스 상태
- 카드: 콘텐츠 박스 디자인

## 📦 기술 스택
- Vue 3 + `<script setup>`
- Tailwind CSS
